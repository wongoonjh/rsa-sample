# RSA sample
- javascript 암호화후, 서버에서 복호화처리
1. 키생성
  client -> 서버요청 : /ss/step1
2. public키로 암호화
  javascript에서 암호화진행
3. 복호화 요청
  client -> 서버요청 : /ss/test_v
  
## Build

```
make
```



## Install
/rsa-sample/src/main/resources/application.properties  
LISTEN port 변경
```
server.port=8881
```


## Run
```
- Run as > String Boot App
- default : 
  main.js http://a.com:8881
```
